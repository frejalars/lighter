# Smart Home Light Controller

A lightweight C++ console application that simulates controlling apartment lights from a mobile phone.

## Overview

Smart Home Light Controller manages lights in different rooms. Users can switch lights on or off, adjust brightness, and generate a status report.

The project demonstrates object-oriented programming, STL containers, modular architecture, and clean C++ design.

## Features

- Control lights in multiple rooms
- Turn lights on/off
- Adjust brightness
- Display apartment status
- Modular code structure
